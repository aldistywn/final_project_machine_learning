# Sistem Deteksi Penyakit Jantung Menggunakan Machine Learning

## Latar Belakang
Penyakit jantung merupakan salah satu penyebab utama kematian sehingga
diperlukan sistem pendukung untuk membantu proses deteksi dini berbasis data.

## Dataset
- Sumber: UCI Machine Learning Repository
- Jenis data: Dataset medis terstruktur

## Pengolahan Data
- Data cleaning dan encoding
- Penanganan ketidakseimbangan data
- Pembagian data latih dan data uji

## Metode Analisis
- Decision Tree
- XGBoost
- Perbandingan performa antar model

## Evaluasi
- Accuracy, precision, recall, dan confusion matrix
- Analisis performa model berdasarkan hasil prediksi

## Kesimpulan & Insight
Hasil analisis menunjukkan bahwa penanganan ketidakseimbangan data berperan
penting dalam meningkatkan kemampuan model mendeteksi kasus positif.
Perbandingan model membantu menentukan metode yang paling sesuai untuk
konteks data medis.
